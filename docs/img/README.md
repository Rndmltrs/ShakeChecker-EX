# Landing-page images

Drop screenshots/demo frames here so the "See it in action" section on the landing
page fills in. Until they exist, those `<img>`s hide themselves (the page stays
clean).

Expected files (referenced by `docs/index.html`):

- `overlay-battle.png` — the catch-rate overlay during a wild encounter
- `dex-panel.png` — the "missing here" dex panel in the overworld

Tips:

- A short animated **GIF** works in an `<img>` too — e.g. save `overlay-battle.png`
  as a GIF of a battle to show it live (just keep the same filename, or add a new
  `<img>` in `docs/index.html`).
- Crop to the game window and keep them a similar aspect ratio so the grid lines up.
- PNG or GIF; aim for < ~2 MB each so the page loads fast.
