ShakeChecker

A live catch-rate overlay for PokeMMO. See the exact catch % for every Poke Ball during wild battles — read straight off your screen.

Windows 10/11 · free & open source

Strictly read-only & safe. ShakeChecker only captures the screen and draws an overlay. It never sends inputs, never reads game memory, never modifies the client. A PokeMMO staff member confirmed that OCR + overlays with no game inputs are fine and not against the rules.

What it does

Live catch rates

Every Poke Ball's catch % recomputed each frame from the enemy's HP, status and turn — using PokeMMO's actual catch formula.

Smart ball logic

Quick, Timer, Nest, Net, Dusk (cave & night), Dream (sleep turns) and Repeat (catch chain) are all modelled correctly.

Hordes & doubles

Waits for a single catchable Pokemon to remain and reads its status correctly even as a horde thins out.

"Missing here" dex

Reads your location from the HUD and lists the species you still need on that route — per PokeMMO account.

Stays out of the way

A click-through overlay docked to the game window. Input passes straight through to the game underneath.

Any window size

Works at any resolution or aspect ratio — even if you resize the game window in the middle of a battle.

Getting started

  1. Download the latest release zip and extract it anywhere.
  2. Run ShakeChecker.exe.
  3. First launch only: Windows SmartScreen may say "Windows protected your PC" → More info → Run anyway (the app is unsigned).
  4. Play PokeMMO in windowed or borderless mode (English client). The overlay appears in battles.

FAQ

Is this allowed? Will I get banned?

A PokeMMO staff member confirmed that "OCR and overlays that make no game inputs are fine and not against the rules." ShakeChecker is exactly that: it reads the screen and shows an overlay, and never sends any input to the game.

Is it a bot? Does it automate anything?

No. It never presses keys or clicks into the game. You play and throw balls yourself — it only displays information. There is no automation of any kind.

Does it read game memory or modify the client?

No. It captures pixels from the screen (the same thing you see) and draws a separate overlay window. It does not read process memory, inject code, or change any game file.

Why does Windows say "Windows protected your PC"?

The app is unsigned (code-signing certificates are expensive for a free tool), so SmartScreen warns on first launch. Click More info → Run anyway. The full source is on GitHub if you'd rather build it yourself.

Does it connect to the internet or collect data?

No. It makes no network connections. Your per-account caught lists are stored locally in %APPDATA%\ShakeChecker and never leave your machine.

My antivirus flagged it.

Apps bundled with PyInstaller sometimes trigger false positives. ShakeChecker is open source — you can read every line and build it yourself from the repo if you prefer.

Does it work at my resolution / window size?

Yes. All detection is based on fractions of the window, not fixed pixels, so it works at any size or aspect ratio — and keeps working if you resize mid-battle.

Mac or Linux?

Windows 10/11 only for now.

How do I update?

Download the newest release and replace the old folder. Your caught lists in %APPDATA%\ShakeChecker are kept.
