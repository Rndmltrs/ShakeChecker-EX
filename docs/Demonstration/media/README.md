# Demo videos

Drop your demo clips here as MP4 (H.264) and list them in `docs/index.html` in the
`VIDEOS` array near the bottom:

```js
const VIDEOS = ["media/demo-1.mp4", "media/demo-2.mp4", "media/demo-3.mp4"];
```

- Any number of clips works — the carousel builds arrows + dots automatically.
- Keep filenames matching the array (or edit the array to match your filenames).
- MP4 / H.264 plays in every browser. Aim for a reasonable size (a 1080p clip of a
  minute or two is usually fine; trim if it gets large).
- Optional: a `docs/img/poster.png` is shown as the video thumbnail before play.
