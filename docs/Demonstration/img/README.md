# Landing-page images

Drop enlarged screenshots here for the "A closer look" section. Each one hides
itself if the file is missing, so add as many or as few as you like:

- `shot-overlay.png` , the catch-rate overlay during a wild battle
- `shot-dex.png` , the Location Dex panel in the overworld
- `shot-status.png` , the overlay reading a status condition
- `shot-horde.png` , the overlay during a horde

Also optional:

- `poster.png` , a thumbnail shown on the demo videos before they play.

Tips: crop to the game window, PNG keeps the pixel art crisp, and clicking a shot
on the page opens the full-size file.
